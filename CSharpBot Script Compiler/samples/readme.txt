Run CSBc.exe with a .csb file as the arguement (e.g. "CSBc.exe helloworld.csb") to compile it.
Run CSBc.exe with an .exe console file as the arguement (e.g. "CSBc.exe helloworld.exe") to test the file.
Run CSBc.exe without an arguement to open the GUI.
